Mechanism: per-query GPU staging area populated by CALL gpu_preload_region(). enc_region skips per-task H2D; dec_region also skips decode but pays cudf::concatenate per scan batch. Aggregate SF=20 sum-of-warm-medians: native 2950ms, enc_region 2030ms (1.45x), dec_region 1981ms (1.49x). H2D bytes drop ~150x (enc) / ~3000x (dec) vs native. Sync time dominates kernel time everywhere (sync/k = 3-9x), confirming TPC-H on Sirius is sync-bound, not transfer-bound.
